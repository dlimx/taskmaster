About Taskmaster

The simple app for organizing your life.

Features

  • Simple way to save your tasks
  • Mark tasks as complete or incomplete
  • Track the dates of your completed tasks
  • Keep up with what you've finished
  • Persistent history tied to your browser

Technical Details

  • Front end logic done with the latest in JavaScript
  • Designed and made using Bulma
  • Saving tasks handled by LocalStorage API

Roadmap

  • Add extra details onto tasks - subtasks, location, assignments
  • Add organization-level features - lists, folders, categories
  • Add reminders and calendar tracking
  • Add server-tracked user-level persistence
  • Migrate to React
  • Migrate to Lambda serverless architecture backed by DynamoDB
